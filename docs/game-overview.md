# Game Overview

## Working title
Charge Runner

## Pitch
A fast, simple mobile lane runner where the player dodges obstacles, collects charge dots, and must build enough charge to pass a barrier at the end of each segment.

## Core loop
- move through lanes
- collect dots
- avoid obstacles
- reach barrier
- pass if charge is high enough
- gain bonus score for extra charge
- continue to a harder segment
- fail if barrier requirement is not met or the player dies
